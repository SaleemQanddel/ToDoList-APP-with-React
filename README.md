# TodoList App

To-Do List Project is a simple application that allows users to manage their daily tasks by adding, editing, deleting, and marking tasks as completed, helping improve time management and productivity 
built using React and Vite.

## Technologies
- React
- Vite
- Docker
- git
## Run with Docker

```bash
docker build -t todo-app .
docker run -p 5173:5173 todo-app
